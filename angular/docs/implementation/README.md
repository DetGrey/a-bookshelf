# Angular Implementation Backlog (Local)

- Source docs: `../ANGULAR_ARCHITECTURE.md`, `../FRONTEND_BLUEPRINT.md`, `../FRONTEND_IMPLEMENTATION_PLAN.md`, `../FRONTEND_REPLICATION_CHECKLIST.md`

- Dependency notation: `ISSUE-XYZ`

## Files

- `issues/ISSUE-001-...md` etc. — implementation tickets in dependency order
